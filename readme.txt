A tool to encrypt your video with an key file and encoded audio and images.
When it is encrypting,do not close the program.It's slow.
The cache images is SUPER large.
Usually,The output file will be multiple times larger. 
You shouldn't encrypt a video over 150 Megabytes.
Be sure you have free space over 1.5G
To use this script,please download ffmpeg first or put the binary to this directory.
